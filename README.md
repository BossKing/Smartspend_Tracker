# SmartSpend AI - Expense Tracker

A modern expense tracking app with AI insights, built with Next.js and Tailwind CSS.

## Quick Start

\`\`\`bash
npm install
npm run dev
\`\`\`

Visit http://localhost:3000

## Features
- Track expenses in INR or USD
- Beautiful interactive charts
- AI-powered insights
- Dark/Light mode
- Fully responsive

## Deploy to Vercel

\`\`\`bash
git push
\`\`\`

Then deploy from vercel.com
